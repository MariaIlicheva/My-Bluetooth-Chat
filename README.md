# Bluetooth Chat App

A simple Bluetooth chat application built with Android Studio using Bluetooth Classic (RFCOMM).

## Features

- Discover nearby Bluetooth devices
- Connect to paired or discovered devices
- Send and receive text messages in real-time
- Supports Android 6.0 and above

## Tech Stack

- **Language**: Java
- **Framework**: Android SDK
- **Protocol**: Bluetooth Classic (RFCOMM)
- **Architecture**: Client-Server (one device acts as server, another as client)
